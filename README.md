# 概述

echo-url 是开箱即用的回显 HTTP 请求的 Serverless 应用. 

路由: 
- /edit 为编辑内容静态页
- /echo 响应数据

# Overview

echo-url is a Serverless application that echoes HTTP requests out of the box.

Routing:
- /edit is a static page for editing content
- /echo is for response data
